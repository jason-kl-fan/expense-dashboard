# 哈狗財務助理

個人消費記帳與統計網站。

## 功能
- 前台快速輸入消費
- 金額按鈕鍵盤，支援小數點
- 類別與付款方式管理
- 後台密碼登入
- 可修改管理密碼
- 日 / 週 / 月 / 年 / 自訂區間統計
- 類別累積與明細編輯

## Firebase 專案
- projectId: `husky-finance-assistant`
- 文件位置: `expenseDashboard/main`

## 注意
這是快速展示版本，後台密碼採簡化做法，資料寫在 Firestore。
正式上線建議改成 Firebase Authentication 與更嚴格的 Firestore Rules。
